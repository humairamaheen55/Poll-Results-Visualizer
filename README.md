# 📊 Poll Results Visualizer

An interactive data analytics project that collects, analyzes, and visualizes survey or poll responses using Python, Pandas, Seaborn, and Streamlit.

---

## 🚀 Project Overview

The Poll Results Visualizer helps transform raw survey responses into meaningful insights through:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Interactive Dashboard
- Insight Generation

This project simulates how organizations analyze survey responses for decision-making.

---

## 🎯 Objectives

- Import poll/survey data from CSV
- Clean and preprocess responses
- Analyze tool preferences and satisfaction ratings
- Generate visual reports
- Deploy results in an interactive Streamlit dashboard

---

## 🛠 Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
- Streamlit
- Jupyter Notebook

## 📈 Features

### 1. Data Cleaning
- Removes duplicates
- Handles missing values
- Converts timestamps
- Standardizes text values

### 2. Data Analysis
- Preferred tool counts
- Satisfaction rating analysis
- Feedback length analysis

### 3. Visualizations
- Bar charts
- Histograms
- Word clouds
- Comparative charts

### 4. Interactive Dashboard
Built using Streamlit with:
- Sidebar filters
- Interactive charts
- Summary insights
